\noindent
При этом, использование кода, загружаемого с серверов «\emph{GeoGebra}», может накладывать некоторые дополнительные ограничения.

Проект «\emph{Geometry Room}» может работать автономно без участия серверов «\emph{GeoGebra}» при определенной настройки.
Но в исходном коде программы «\emph{GeoGebra}» имеются необходимые для работы скрипты с отметкой о несвободной
лицензии, хотя это недопустимо, т.к. весь код поставляется под лицензией «GNU General Public License v3.0» и,
помимо всего прочего, в нем используются зависимости с этой же лицензией.
